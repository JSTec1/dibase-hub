# Coloque as imagens aqui

Para o site mostrar **logo** e **fundo**, estes arquivos precisam estar **dentro de `public/`** antes do build:

- **`logo.png.jpeg`** – logotipo circular (Mineração Dibase)
- **`background.jpg.jpeg`** – foto aérea da mina para o fundo

Depois rode `npm run build` de novo e faça o deploy da pasta **`out`** (veja DEPLOY.md na raiz do projeto).

Produtos (opcional): crie a pasta **`public/produtos/`** e coloque **1.jpg**, **2.jpg**, … **6.jpg**.
